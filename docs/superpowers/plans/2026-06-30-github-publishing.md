> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the existing SNOM Explorer repository directly runnable, documented, and safe to publish on GitHub.

**Architecture:** Keep the flat application layout and launch `app.py` directly. Replace references to the unavailable `stroboscopicsnom_local` package with imports from the local modules; no packaging layer is needed for GitHub distribution.
